\frametitle{TAEN: Tikhonov Autoencoder Network}

{\small

For linear problems, i.e., $\G$ is a linear map, applying 1st optimality condition, and \myblue{\bf if $\bar{Y}$ is a full row rank}, we have 
\begin{itemize}
    \item given a test observation sample $\ybtest$, \myred{test inverse solution error}
        \[
            \epsb_{\ubtest}^{\pureOPO} = \nor{\Psie^*(\ybtest) - \ubtest }_2^2 \leq  \nor{{\ub^\text{test} - {\ub}_0} }_2^2
        \]
    \item given a test parameter $\ubtest$ \myred{test forward solution error}
        \[
            \epsb_{\ybtest}^{\pureOPO} = \nor{\Psid^*(\ubtest) - \ybtest }_2^2  = 0, 
        \]
\end{itemize}
